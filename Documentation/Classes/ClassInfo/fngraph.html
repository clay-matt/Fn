void FnGraph::addEdge ( const QString & edge, const QVector<QString> & endpoints )

The function adds an edge named edge to the FnGraph with initial vertex endpoints.at(0) and terminal vertex endpoints.at(1). These vertices are added to the FnGraph if necessary. If an edge with name edge already exists, it is redefined to have initial and terminal vertices defined by endpoints.

void FnGraph::addEdge ( const QString & edge, const QString & initial, const QString & terminal )

This function overloads FnGraph::addEdge ( const QString & edge, const QVector<QString> & endpoints ).

The function adds an edge named edge to the FnGraph with initial vertex initial and terminal vertex terminal. These vertices are added to the FnGraph if necessary. If an edge with name edge already exists, it is redefined to have initial and terminal vertices initial and terminal respectively.

void FnGraph::addVertex ( const QString & vertex )

This function adds an isolated vertex named vertex to the FnGraph.

QStringList FnGraph::edgeList ( ) const

Returns keys( ), the list of edges in the FnGraph.

int FnGraph::eulerCharacteristic ( ) const

Returns the Euler Characteristic of the FnGraph, i.e., the number of vertices minus the number of edges.

QString FnGraph::initialVertex( const QString & edge ) const

Returns the name of the initial vertex of edge. If edge is not an edge of the FnGraph, an empty QString is returned.

int FnGraph::nEdges ( ) const

Returns size( ), the number of edges in the FnGraph.

int FnGraph::nVertices ( ) const

Returns vertices.size( ), the number of vertices in the FnGraph.

QString FnGraph::terminalVertex( const QString & edge ) const

Returns the name of the terminal vertex of edge. If edge is not an edge of the FnGraph, an empty QString is returned.

QStringList FnGraph::vertexList ( ) const

Returns vertices.keys( ), the list of vertices in the FnGraph.
